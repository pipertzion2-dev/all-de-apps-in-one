Stripe Payment Integration
  • Create Stripe account Required
    Go to stripe.com and create a free account if you don't have one
  • Get Stripe API keys
    In Stripe Dashboard → Developers → API keys. Copy the Secret key (starts with sk_)
  • Add STRIPE_SECRET_KEY to Secrets
    In Replit: Click "Secrets" tab (lock icon) → Add STRIPE_SECRET_KEY with your secret key
  • Create pricing plans in Stripe
    Stripe Dashboard → Products → Create products for your subscription tiers (e.g., Free, Pro, Enterprise)
  • Set up Stripe webhook (for production)
    Stripe Dashboard → Webhooks → Add endpoint pointing to your published app URL + /api/stripe/webhook
Tip: Test Mode
Use Stripe's test mode keys (starting with sk_test_) while developing. Switch to live keys only when ready to accept real payments.

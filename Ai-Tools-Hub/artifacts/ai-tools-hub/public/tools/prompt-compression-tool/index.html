What this tool does

It estimates token usage, finds redundant or verbose phrasing, and suggests a shorter version of your prompt. Use it to cut cost and latency before calling AI or API endpoints.

Compression score

0–100: higher means more of the prompt could be shortened without losing meaning (repetition, filler, long instructions).

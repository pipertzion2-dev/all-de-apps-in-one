How to read this

Compatibility score — How well the model handles structured output when you provide a schema. Higher is better for strict schemas.

Est. tokens — Approximate input + output tokens per call. Use this to compare cost and rate limits.

Est. cost per call — Rough cost in USD for one request at current list prices. Useful for budgeting before switching models.

Schema-friendly — Indicates whether the model is a good fit when you enforce a JSON schema or similar.

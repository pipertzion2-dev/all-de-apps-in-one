      Guidance

      What is prompt consistency?

      Consistency means using the same terms for the same concepts, keeping a clear structure, and aligning your prompt with your schema or API so the model behaves predictably.

      How we score

      We evaluate terminology (repeated use of the same words for the same idea), structure (sections, bullets, numbering), clarity (sentence length), and—if you provide a schema—how well the prompt references your fields.

      Improving your score

      Use one term per concept, add clear headings or steps, keep instructions between 10–25 words per sentence, and mention key schema field names in the prompt.

      Schema alignment

      Paste a JSON schema or API definition in the left panel. We check that your prompt refers to those field names so the model is more likely to output valid, structured data.

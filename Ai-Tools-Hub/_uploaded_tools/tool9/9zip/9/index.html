Common JSON issues

  • Trailing commas — JSON does not allow a comma after the last element in an array or object.
  • Unquoted keys — All object keys must be double-quoted strings.
  • Single quotes — Use double quotes (") for strings.
  • Comments — Standard JSON has no // or /* */; strip them before parsing.

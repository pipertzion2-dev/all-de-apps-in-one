# Hardware Risk & Compliance Scanner (Mini App #26)

Free, no-login preview of one step of Svivva’s hardware pipeline: risk & compliance scan from a short brief.

- **Target keywords:** hardware risk & compliance scanner tool, free, online, AI
- **Layout:** Three-panel (inputs | preview | workflow mapping), zc regular font, accent `#B45309`
- **Primary CTA:** Try Svivva free (sticky bottom bar)

## Run

```bash
npm install
npm run dev
```

Build: `npm run build`. Preview build: `npm run preview`.

## Font

Zc Regular is in `public/fonts/Zc-Regular.ttf`. If missing, copy from `../21/dist/fonts/Zc-Regular.ttf`.

## Analytics

Events (when `gtag` is present): `preview generated`, `CTA click`, `module interest`.

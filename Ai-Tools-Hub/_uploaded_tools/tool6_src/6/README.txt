PROMPT DRIFT DETECTOR — HOW TO OPEN
====================================

WAY 1 — No server (open the file)
---------------------------------
• In Finder, open this folder (6).
• Double-click:  index.html
  Safari (or your default browser) will open the app.
  The address bar will show something like: file:///Users/.../index.html
  That’s normal. The app works.

If Safari doesn’t open it: in Safari choose File → Open File… and select index.html from this folder.


WAY 2 — With a URL (http://localhost:5999)
------------------------------------------
• Double-click:  Start Server.command
• A Terminal window will open and show a message.
• Open Safari. In the address bar type exactly:  http://localhost:5999
• Press Enter.
• Leave the Terminal window open while you use the app.
• When done, close the Terminal window to stop the server.

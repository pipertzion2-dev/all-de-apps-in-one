# Open Prompt Drift Detector in Safari (no server)

You don’t need a server. Open the file directly:

## Steps

1. In **Finder**, go to:  
   **Documents → 40 Cursor Apps → 6**

2. Double‑click **index.html** or **prompt-drift-detector.html**

   **Or** in Safari: **File → Open File…** and choose that file from folder **6**.

3. The app runs in Safari. You should see the purple bar: **Mini App #16 — Prompt Drift Detector**.

Everything runs in the browser; nothing is sent to any server.

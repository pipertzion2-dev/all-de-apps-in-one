# Put the app on a real server (free, works in any browser)

## Option 1 — Netlify Drop (easiest, no account required to try)

1. Open **https://app.netlify.com/drop** in your browser.
2. Drag the **entire folder** `6` (this folder) onto the page.
3. Netlify will give you a URL like **https://random-name-12345.netlify.app**.
4. Open that URL in Safari (or any device). The app will work normally.

You can create a free Netlify account to keep the URL and update the site later.

---

## Option 2 — Local server on port 8080

1. Double-click **Start Server.command** in this folder.
2. When Terminal opens, Safari should open to **http://127.0.0.1:8080**.
3. Keep the Terminal window open while you use the app.

If Safari doesn’t open, go to **http://127.0.0.1:8080** yourself.
